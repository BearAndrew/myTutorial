object-fit

  • fill
    預設值，預設會強制變形至 css 所定義的元素寬高，不管原檔比例。
  • contain
    增加或減少載入物件的寬度及高度(維持原比例)直到放得進所定義的元素寬高。
  • cover
    填滿元素的寬度及高度(維持原比例)，通常會剪掉部分的物件。
  • none
    不做任何大小及比例調整。
  • scale-down
    將會選擇設為 none 或 container 兩者間較小的那個物件。

範例

img
height: 150px
object-fit: [option]
