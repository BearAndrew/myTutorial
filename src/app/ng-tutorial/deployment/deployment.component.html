Deployment

IIS

1. 在終端機輸入下列指令打包專案

ng build --prod --base-href /[app-name]/

2. 在IIS中，站台底下的網站，按右鍵新增應用程式，應用程式名稱輸入 [app-name]

3. 實體檔案位置選擇 Angular app 資料夾中的 dist / [app-name]

4. 新增文字文件，複製下方 code ，修改 <action> 中 url ，並將文字檔更名為 web.config，之後放入實體檔案位置的資料夾

Github Page

參考資料

1. 安裝 angular-cli-ghpages 套件

npm i -g angular-cli-ghpages

2. 在終端機輸入下列指令打包專案

ng build --prod --base-href /[app-name]/

3. 將專案 push 至 github

git remote add origin YOUR_GITHUB_URL

4. 發佈至 github

npx ngh --dir=dist/[app-name] --no-silent

5. 瀏覽頁面

https://YOUR_GIT_ACCOUNT.github.io/YOUR_APP_NAME
